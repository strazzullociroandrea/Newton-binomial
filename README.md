# NEWTON'S BINOMIAL

Newton's Binomial è una web-app moderna progettata per automatizzare il calcolo e la visualizzazione dello sviluppo del binomio **(a+b)^n**. Grazie all'implementazione del Teorema del Binomio di Newton, l'applicazione genera l'espansione polinomiale completa, gestendo con precisione i coefficienti binomiali e la formattazione algebrica delle potenze.

## Caratteristiche principali
- **Espansione dinamica**: Calcolo di ogni termine dello sviluppo polimoniale, adattandosi all'esponente inserito

## Stack tecnologico
Il progetto è stato sviluppato utilizzando:
- Frontend e UI: React, Tailwind CSS, Shadcn UI
- Backend: tRPC
- Testing: Vitest
